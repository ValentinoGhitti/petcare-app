# Pet Care App

## Descripción

El dashboard de PetCare es una aplicación diseñada para proporcionar una interfaz amigable y funcional para el software de gestión de mascotas. La aplicación está desarrollada con Vue 2 y utilizando la librería de componentes de Vuetify, siguiendo las últimas directrices de UX/UI proporcionadas. Utilicé vuex como gestor de estado para la persistencia de datos en la app. Datos de los usuarios, credencial de inicio de sesión todo se almacena en localStorage. También usé la store para que maneje el estado de autenticación, inicio de sesión, cierre de sesión, registro de usuario y restablecimiento de contraseña.
Para el tema de gestión de rutas y navegación use vue router; para acceder al dashboard de la aplicación requiere autenticación.
